POC to spinup `localstack` and some deployment tools (i.e. `aws-cdk`) - to have testable playground for infrustructure.

### Before start 
#### Prerequisites
* install [`nix`]()
* install [`direnv`]()
* install [`lorri`]()

after that `lorri shell` or `nix-shell` and you are ready to go!
